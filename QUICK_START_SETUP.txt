⚡ COMPLETE THE SETUP IN 2 STEPS (5 minutes total):

STEP 1: Configure Railway Backend
─────────────────────────────────────────────────────────────────

1. Visit: https://railway.com/dashboard
2. Select: "job-portal-backend" project
3. Click the SERVICE CARD
4. Go to: Settings (tab) → Environment Variables
5. Click: "Add Variable" and add these 3:

   KEY: MONGODB_URI
   VALUE: mongodb+srv://agvansh:vansh123@jobportal.efztav4.mongodb.net/?appName=jobportal

   KEY: JWT_SECRET
   VALUE: your_jwt_secret_key_here_change_in_production

   KEY: PORT
   VALUE: 8080

6. Click "Save Variables"
7. Wait for "Redeploy" button to appear (top right)
8. Click "Redeploy"
9. Wait ~1-2 minutes for deployment
10. 📍 COPY the URL from the top of the page 
    (looks like: https://job-portal-xxxxxx.up.railway.app)


STEP 2: Update Vercel with Backend URL
─────────────────────────────────────────────────────────────────

1. Visit: https://vercel.com/dashboard
2. Select: "job-portal" project
3. Go to: Settings (tab) → Environment Variables
4. Find or create variable:
   KEY: VITE_API_URL
   VALUE: [PASTE YOUR RAILWAY URL]/api
   Example: https://job-portal-production.up.railway.app/api
5. Select "Production" environment
6. Click "Save"
7. Go to "Deployments" tab
8. Click "Redeploy" on the latest deployment
9. Wait ~1-2 minutes
